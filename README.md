vim
===

my .vim folder

Usage
=====

	mkdir ~/.vim
	cd ~/.vim
	git remote add https://github.com/jcfrank/vim.git
	git pull origin master
	./cloneVundle.sh

After these, launch vim and run:

	:BundleInstall

Finally, exit vim and execute script:

	./removeDupSnippets.sh
